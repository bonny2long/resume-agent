# United Airlines AI Insights Platform

**Role**: Full Stack Software Engineer Intern  
**Timeline**: 3 months (Oct 2024 - Dec 2024)  
**Team Size**: Solo developer at i.c Stars  
**Client**: United Airlines (through competitive RFP process)

## Project Overview
AI-powered customer feedback analysis platform that transforms unstructured feedback into actionable insights for different organizational levels (executives, analysts, administrators).

## What I Built

### Core Architecture
- **Frontend**: React with TypeScript, Tailwind CSS
- **Backend**: Node.js, Express.js
- **Database**: PostgreSQL via Supabase
- **AI Layer**: LLM pipelines using OpenAI/Claude APIs
- **Authentication**: Role-based access control (RBAC)

### Key Features
1. **Feedback Processing Pipeline**: Automated analysis of customer feedback using custom prompts
2. **Multi-Role Dashboard**: Different views for executives, admins, and analysts
3. **Sentiment Analysis**: Real-time sentiment scoring and trend identification
4. **Report Generation**: Automated insights and recommendations

## Challenges Overcome

### 1. LLM Cost Management
**Problem**: High API costs for processing large feedback volumes
**Solution**: Implemented intelligent batching and caching strategies
**Result**: Reduced costs by 40% while maintaining quality

### 2. Enterprise Security Requirements
**Problem**: United Airlines required strict security protocols
**Solution**: Multi-layered security with RBAC, encryption, and audit logging
**Result**: Passed security review and gained client approval

### 3. Multi-Role Complexity
**Problem**: Different user types needed different data views
**Solution**: Flexible architecture with role-based components
**Result**: Three distinct user experiences with shared backend
